## 🚀 البدء السريع

### المتطلبات:
- Python 3.11+
- FFmpeg
- مفتاح Telegram Bot

### التثبيت:

```bash
# استنساخ المستودع
git clone https://github.com/nguf624-a11y/telegram_downloader_bot.git
cd telegram_downloader_bot

# تثبيت المكتبات
pip install -r requirements.txt

# تشغيل البوت
python bot.py
```

## 🌐 الاستضافة على Railway

### الخطوات:

1. **إنشاء حساب Railway:**
   - اذهب إلى [Railway.app](https://railway.app)
   - سجل بحسابك في GitHub

2. **نشر المشروع:**
   - اضغط "New Project"
   - اختر "Deploy from GitHub"
   - اختر هذا المستودع

3. **إضافة متغيرات البيئة:**
   - لا توجد متغيرات بيئة مطلوبة (المفتاح موجود في الكود)

4. **التشغيل:**
   - Railway سيشغل البوت تلقائياً

---

## 🔔 المراقبة مع Uptime Robot

### الخطوات:

1. **إنشاء حساب Uptime Robot:**
   - اذهب إلى [UptimeRobot.com](https://uptimerobot.com)
   - سجل مجاناً

2. **إضافة Monitor:**
   - اضغط "Add Monitor"
   - اختر "HTTP(s)"
   - أدخل رابط البوت من Railway
   - اضبط التكرار كل 5 دقائق

3. **التفعيل:**
   - Uptime Robot سيراقب البوت 24/7
